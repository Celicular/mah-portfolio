## 🚀 Vision
This portfolio is designed to "WOW" viewers at first glance. It bridges the gap between enterprise-grade engineering and experimental artistic design, utilizing modern web technologies to create a living, breathing digital identity.

## 🛠 Tech Stack

### Core
- **React 19**: The foundation for a component-based, scalable architecture.
- **Vite 8**: Next-generation frontend tooling for lightning-fast development and optimized production builds.
- **Tailwind CSS 4**: A utility-first CSS framework for rapid UI development and a cohesive design system.

### Animation & Experience
- **GSAP (GreenSock Animation Platform)**: Handles complex scroll-triggered animations and high-performance layout transitions via `ScrollTrigger`.
- **Framer Motion**: Powering subtle micro-animations, text morphing, and page transitions for an "alive" feel.
- **Lenis**: Implementation of smooth scrolling to provide a premium, unified scrolling experience across all browsers and devices.
- **Lucide React**: A suite of clean, consistent icons for a modern interface.

### Design System
- **Glassmorphism**: Custom utilities for blurred, translucent backgrounds and frosted glass borders.
- **Dynamic Glows**: SVG and CSS-based radial gradients that pulse and move based on user interaction or scroll position.
- **Responsive-First**: Separate desktop (GSAP-heavy) and mobile (performance-optimized) layouts to ensure a flawless experience on any screen size.

## ✨ Key Features
- **Interactive Hero Section**: Dual portraits with parallax motion and perspective-shifted typography.
- **Smooth Navigation**: Auto-smooth-scroll implementation for all navbar links.
- **About Me Morphing**: Text-switching logic synchronized with scroll progress.
- **Professional Portfolio Slider**: A horizontal image slider for major enterprise projects.
- **Personal Projects Workspace**: A hybrid sticky-scrolling layout showcasing open-source contributions.
- **Custom Hardware-Accelerated Cursor**: Enhances the desktop experience with subtle follow-delay and interaction feedback.
- **Full-Color Imagery**: Optimized local assets (`celi1`, `celi2`, `celi3`) integrated seamlessly into the design.

## 🏗 Installation & Development

1. **Clone the repository:**
   ```bash
   git clone <your-repo-url>
   cd portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
